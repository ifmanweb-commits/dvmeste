## 📋 ТЗ: Раздел модерации документов психологов

### 1️⃣ Общая информация

**Проект:** Давай вместе — каталог психологов
**Технологии:** Next.js 16 (App Router), Prisma, Tailwind CSS
**Тип задачи:** Создание раздела модерации документов (дипломы, сертификаты, курсы)

### 2️⃣ Текущее состояние

#### Модель Document
```prisma
model Document {
  id           String       @id @default(cuid())
  type         DocumentType // ACADEMIC_EDUCATION, PROFESSIONAL_TRAINING, COURSE
  url          String       // путь к файлу
  filename     String
  mimeType     String       // image/jpeg, image/png, application/pdf
  size         Int
  description  String?      // пользовательский комментарий
  userId       String
  verifiedAt   DateTime?    // null = не проверено
  verifiedBy   String?      // кто проверил
  uploadedAt   DateTime     @default(now())
  organization String?      // учебное заведение
  programName  String?      // название программы
  year         Int?         // год окончания
  
  user         User         @relation("user_documents")
}

enum DocumentType {
  ACADEMIC_EDUCATION  // диплом о высшем
  PROFESSIONAL_TRAINING // профпереподготовка
  COURSE              // курс
  SUPPORTING_DOC      // подтверждающий документ
  LINK                // ссылка
  PHOTO               // фото (отдельно)
  OTHER
}
```

#### В. Детали отображения

**Для изображений:**
- Превью на всю ширину карточки (с сохранением пропорций)
- При клике на превью — открывается в новой вкладке (через `target="_blank"`)

**Для PDF:**
- Большая иконка PDF (из lucide-react: `FileText` или `File`)
- Крупное название файла под иконкой
- При клике на иконку или кнопку "Открыть" — открывается в новой вкладке

**Мета-информация под превью:**
- Организация (если есть)
- Год окончания (если есть)
- Название программы (если есть)
- Имя файла и размер
- Тип документа: 
  - Академическое образование
  - Профессиональная подготовка
  - Курсы и интенсивы
  - Другие документы

#### Г. Логика действий

**Принять документ:**
- Устанавливает `verifiedAt = new Date()`
- Документ учитывается в профиле психолога
- Исчезает из списка модерации

**Отклонить документ:**
1. Показать модальное окно подтверждения: "Удалить документ безвозвратно?"
2. При подтверждении:
   - Удалить файл с диска (`fs.unlink`)
   - Удалить запись из таблицы Document
3. При успехе — документ исчезает из списка

**Кнопки:**
- `[🔗 Открыть в новом окне]` — для всех типов файлов
- `[✅ Принять]` — зеленая кнопка
- `[❌ Отклонить]` — красная кнопка

#### Д. Фильтры
- Выпадающий список с типами документов
- По умолчанию: "Все"

#### Е. Пустые состояния
- Если нет документов на модерацию — показывать заглушку
- Если у пользователя нет документов — не показывать в списке
